Input-Typen

<input type="text">

Das <input type="text">-Tag wird verwendet, um Texteingabefelder zu erstellen.

<input type="password">

Das <input type="password">-Tag wird verwendet, um Passworteingabefelder zu erstellen.

<input type="checkbox">

Das <input type="checkbox">-Tag wird verwendet, um Kontrollkästchen (Checkboxen) zu erstellen.

<input type="radio">

Das <input type="radio">-Tag wird verwendet, um Optionsfelder (Radio-Buttons) zu erstellen.

<input type="submit">

Das <input type="submit">-Tag wird verwendet, um Schaltflächen zum Absenden von Formularen zu erstellen.
